Dev-Bot:

  • It is an interactive quiz chatbot designed to enhance learning by generating quizzes based on user-specified topics. After receiving a topic, Dev-Bot creates a quiz with 4-5 questions to test and reinforce the user's knowledge.

  • Dev-Bot implements Retrieval-Augmented Generation (RAG) using the Wikipedia API to fetch related information, ensuring quiz content is up-to-date and informative. This feature enriches the quiz experience by incorporating real-time data. Hosted on Hugging Face Spaces, Dev-Bot uses advanced NLP techniques to ensure quizzes are accurate and relevant, making learning both fun and engaging.
