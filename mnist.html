MNIST Neural Network

Project Information

  • Category: Machine Learning
  • Project URL: Github repo

Implemented a Deep Learning neural network from scratch without any machine learning frameworks using only NumPy and Pandas libraries. The NN has a three-layer architecture. It was trained on the MNIST dataset from kaggle. The neural network was able to acheive 93% accuracy.

Neural Architecture :

  • Input Layer (784 nodes)
  • Hidden Layer 1 (128 nodes)
  • Hidden Layer 2 (10 nodes)
  • Output Layer (10 nodes)
